## Run Locally

Clone the project

```bash
  git clone https://github.com/TheGaurav123/CRUD-UniversityManagement.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
